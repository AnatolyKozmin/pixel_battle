# Звуковые файлы

Разместите здесь звуковые файлы для игры:

## Необходимые файлы:

1. **background-music.mp3** - Фоновая музыка (рекомендуется: 30-60 секунд, зацикленная)
   - Формат: MP3, OGG
   - Длительность: 30-60 секунд (будет зациклена)
   - Громкость: средняя (будет уменьшена до 30%)

2. **pixel-place.mp3** - Звук размещения пикселя
   - Формат: MP3, OGG, WAV
   - Длительность: 0.1-0.3 секунды
   - Тип: короткий "click" или "pop" звук

3. **pixel-error.mp3** - Звук ошибки
   - Формат: MP3, OGG, WAV
   - Длительность: 0.2-0.5 секунды
   - Тип: короткий звук ошибки

4. **achievement.mp3** - Звук достижения
   - Формат: MP3, OGG, WAV
   - Длительность: 0.5-1 секунда
   - Тип: приятный звук успеха

5. **notification.mp3** - Звук уведомления
   - Формат: MP3, OGG, WAV
   - Длительность: 0.3-0.6 секунды
   - Тип: мягкий звук уведомления

## Где найти звуки:

- **Freesound.org** - бесплатные звуки (CC0, CC BY)
- **Zapsplat.com** - бесплатные звуки (требуется регистрация)
- **OpenGameArt.org** - звуки для игр
- **Incompetech.com** - бесплатная музыка (Kevin MacLeod)

## Рекомендации:

- Используйте короткие звуки для лучшей производительности
- Оптимизируйте файлы (сжатие MP3)
- Тестируйте на разных устройствах
- Учитывайте размер файлов (каждый файл < 100KB)

## Генерация звуков:

Можно использовать онлайн генераторы:
- **Bfxr** - для звуков эффектов
- **ChipTone** - для 8-bit звуков
- **Audacity** - для редактирования
